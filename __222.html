    5. Multicast related command‌


      1. AT+JOINGROUP: Join a multicast network‌


        Taixin

        Confidential

        After setting the working mode of the WiFi module to group, this command can be used to set WiFi module to join a multicast network. After joining a multicast network, the WiFi module will only receive the data in the multicast network. All data communications are carried out with multicast addresses. If the working mode is set to group, but does not join the multicast network, all data communications are sent and received in the form of broadcast.

        execution

        instruction


        Set: AT+JOINGROUP=11:22:33:44:55:66,3

        response


        Success: OK

        Fail: ERROR

        parameter specification


        AT+JOINGROUP=group_addr,AID

        group_addr: The address of multicast network to be join. AID：The AID of the device in multicast network，the effective value of AID：1~255. The AID of device in network should be unique.

        example


        AT+JOINGROUP=11:22:33:44:55:66,3

        Join the multicast address:11:22:33:44:55:66 AID set to 3

        • Set effective AID: WiFi module will periodically send heartbeat over the multicast network to announce its presence to other WiFi modules.

        • Set ineffective AID：WiFi module will not send heartbeat, and will not announce other WiFi module.
